# AI・固定ロジック・人間の境界

| 領域 | 担当 |
|---|---|
| 認証、認可、計算、状態遷移、監査 | 通常システム |
| 金額閾値、承認経路、必須条件 | ルールエンジン |
| 文脈、分類、仮説、意味差分、下書き | AI |
| 価値、責任、例外、正式決定 | 人間 |

## 実行レベル

- L0 検索・要約: 自動
- L1 候補・下書き: 自動＋表示
- L2 社内保存・タスク登録: 利用者確認
- L3 正式公開・部門横断更新: 責任者承認
- L4 顧客送信・予算・基準改定: 複数承認
- L5 権限変更・契約・人事判断: 人間のみ

## 思考停止防止

- ExploreとChallengeをDecideより前に選択可能にする
- 一案だけでなく、代替案、反対意見、何もしない案を提示する
- 沈黙を合意として扱わない
- 重要案件では人間の前提を先に記録するThink Firstモードを設ける
