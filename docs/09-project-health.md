# Project Health Intelligence

## 評価領域

戦略・価値、スコープ、進捗、コスト、要員、品質、意思決定、リスク、ステークホルダー、統制、定着、成果。

## 表示単位

状態、背景、影響、傾向、予測、対応状況、判断事項、根拠、確信度、データ欠損をセットで表示する。

## ステークホルダービュー

- 経営: 投資、戦略、転換点、撤退条件
- 管理職: 要員、依存、支援、判断待ち
- 現場: 今日のアクション、変更、ブロッカー
- 顧客: 共同合意、マイルストーン、共有可能なリスク

## エスカレーション

プロジェクト内 → 部門支援 → 部門横断 → 経営判断の順に、影響額、顧客影響、戦略的重要度、解決権限で決定する。
