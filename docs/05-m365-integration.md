# Microsoft 365統合

## 正本

| 情報 | 正本 |
|---|---|
| 人、所属、グループ | Entra ID |
| チーム参加 | Microsoft 365 Groups / Teams |
| メール、予定 | Exchange / Outlook |
| 会議、チャット | Teams |
| 文書 | SharePoint / OneDrive |
| タスク | Planner等 |
| ラベル、保持、DLP | Purview |
| 意思決定、ヘルス、基準構造 | 独自台帳 |

## プッシュ型体験

- Critical: Teams即時通知＋必要時メール
- Action: Adaptive Card / Outlook Actionable Message
- Advisory: ダッシュボード、日次ダイジェスト
- Information: 週次キャッチアップ

同一イベントを複数媒体へ重複通知せず、未処理時だけ段階的にエスカレーションする。

## アクセス

実効アクセス = 利用者権限 ∩ エージェント職務範囲 ∩ 業務目的 ∩ 秘密度ポリシー。
