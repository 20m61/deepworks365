# リスク登録簿

| リスク | 影響 | 主な対策 |
|---|---|---|
| 思考の外注化 | 判断能力・育成の低下 | Think First、Challenge、判断理由記録 |
| AIの結論誘導 | 選択肢の狭窄 | 探索と判断フェーズ分離、複数案 |
| 意見の均質化 | 革新性低下 | ゼロベース案、少数意見保護 |
| 監視社会化 | 発言・相談の萎縮 | 目的制限、メンター領域分離、人物評価禁止 |
| 合意の形骸化 | 未納得の決定 | 条件付き賛成、異論、沈黙非同意 |
| 文脈誤認 | 誤った判断連鎖 | 根拠、確信度、訂正権、正式情報分離 |
| AI生成物の情報汚染 | 正本不明・検索悪化 | 構造化データ、状態、保存抑制 |
| 拙速な判断 | 重要判断の失敗 | 即断可否分類、観察・対話期間 |
| 権限逸脱 | 情報漏えい | Entra委任、職務制限、DLP、監査 |
| プロンプトインジェクション | 誤操作・漏えい | データと命令分離、ツール制限、Hooks |
| 高コスト化 | ROI悪化 | ルーティング、FinOps、休止・廃止 |
| 運用属人化 | 継続不能 | IaC、Golden Path、自動診断、Runbook |
| AI品質劣化 | 誤出力増加 | ゴールデンセット、継続評価、ロールバック |
| 外部情報の誤利用 | 誤判断・権利侵害 | 一次情報、鮮度、引用、承認 |
