# プロジェクト憲章

## プロジェクト名

Organizational Intelligence Platform（仮称）

## 背景

企業内には、会議、メール、チャット、ファイル、タスク、基準、過去事例が大量に存在する。しかし、時系列、判断の背景、正本、担当者ごとの整理意図が分断され、人間が検索、転記、資料作成、進捗集計、督促へ多くの時間を費やしている。

一方、単純なAI自動化は、思考停止、監視社会化、権限逸脱、誤情報の固定化、成果物の大量生成を招く可能性がある。

## ビジョン

人間は人間同士で理想、問い、異論、顧客価値を語り、合意と責任ある判断を行う。システムは、その対話を組織の文脈、基準、実績へ接続し、検証可能な成果物、タスク、ダッシュボード、学習へ変換する。

## 目的

1. 8,000人規模の組織で、情報探索、資料作成、会議後処理、進捗報告の時間を削減する
2. 経営・管理職・現場が同じ事実から役割別に状況を把握する
3. 意思決定の速度、根拠性、再現性、追跡可能性を高める
4. 組織基準と暗黙知を実務から継続的に育てる
5. 少人数で安全・低コストに開発、運用、改善できるプラットフォームを作る

## 設計原則

- 作業摩擦を減らし、思考のための摩擦は残す
- Microsoft 365 / Entra IDを権限と組織情報の正とする
- 固定ロジック、AI推論、人間判断を分離する
- AIの推奨と正式な決定を分離する
- 同じ事実から役割別ビューと成果物を生成する
- 結論だけでなく、背景、異論、不確実性、撤退条件を示す
- 重要操作はプレビュー、差分、承認、取消、監査を持つ
- 個人を採点せず、仕事が進む条件を可視化する
- オブザーバビリティ、セキュリティ、FinOpsを初日から実装する

## 対象範囲

- Teams / Outlook / SharePoint / OneDrive / Planner / Microsoft 365 Copilotとの連携
- 合意・意思決定台帳
- 組織文脈グラフとオントロジー
- 組織基準育成
- プロジェクトヘルス、経営判断キュー、ダッシュボード
- 専門職エージェント
- 構造化成果物とDecision & Review Experience
- 外部連携ゾーン
- 開発・運用統制基盤

## 非目標

- AIによる人事評価や人物ランキング
- AIによる最終決裁、契約締結、権限変更
- Microsoft 365の権限を迂回する独自データ収集
- 全文書の無制限コピーと恒久保存
- すべての業務を単一の万能エージェントへ集約すること

## 成功条件

- 全社平均で1人1日10分以上の有効時間を創出
- 重大なアクセス権逸脱・外部誤送信ゼロ
- 会議終了から初稿作成までを80%以上短縮
- 重要判断の根拠充足率95%以上
- 不要通知率10%未満
- 定常運用を5人相当以下の中核チームで実施可能
- AI機能の重大事実誤りを自動ゲートまたは人間レビューで遮断

## ガバナンス

- Executive Sponsor: 経営目的と投資判断
- Product Owner: 価値、優先順位、成果指標
- Platform Owner: アーキテクチャ、SRE、IaC
- Data / Standards Owners: データ品質、基準
- Security / Privacy: Entra、Purview、AppSec、法務
- Agent Owners: エージェント品質、権限、コスト
- Department Champions: 部門文脈、受入、フィードバック
